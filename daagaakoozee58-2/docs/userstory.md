# Definition of Done (DoD)

Een user story is **Done** wanneer aan alle onderstaande punten is voldaan:

- Alle code is geschreven en **werkt zoals bedoeld**, zonder fouten of crashes.
- Er zijn **tests uitgevoerd** en deze zijn geslaagd.
- De code is **gereviewd door een teamlid**.
- Er is actief **feedback gevraagd**, zodat verbeterpunten besproken konden worden.
- Alle noodzakelijke **bugs of issues** die tijdens de sprint zijn ontdekt, zijn opgelost.
- De functionaliteit voldoet aan alle **acceptatiecriteria** van de user story.
- De code volgt de  **codeconventies van de Knowledge Base (KB)**.
